# Banking App - Java Application

A 3-Tier Full Stack Java Banking Application built with Spring Boot.

---

## 🚀 Features

- User registration & authentication
- Account management (deposit, withdraw, transfer)
- Transaction history
- Secure password handling
- RESTful API architecture

---

## 🛠️ Tech Stack

- **Backend:** Java, Spring Boot
- **Database:** (MySQL)
- **Security:** Spring Security

---

## 📦 Getting Started

1. **Clone the repository:**
   ```bash
   git clone https://github.com/widushan/Banking-App--Java-Application.git
   cd your-repo-name
   ```

2. **Configure the database:**  
   Update `application.properties` with your DB credentials.

3. **Build and run:**
   ```bash
   mvn spring-boot:run
   ```

   http://localhost:8080/
